**CGPA & SGPA Calculator with Report Generator – Mini Administrative System**

## Project Description

The CGPA & SGPA Calculator with Report Generator is a mini administrative system developed to automate the calculation and management of students' academic performance. The system calculates Semester Grade Point Average (SGPA) and Cumulative Grade Point Average (CGPA) based on course grades and credit hours, providing accurate and efficient academic evaluation.

The application allows administrators to manage student records, course details, grades, and credit hours through a user-friendly interface. It automatically computes SGPA for each semester and CGPA across all semesters, eliminating manual calculations and reducing errors.

In addition, the system includes a Report Generator module that creates detailed academic reports containing student information, semester-wise results, SGPA, CGPA, total credits earned, and overall academic standing. These reports can be viewed, printed, or saved for administrative and student reference.

## Objectives

* Automate SGPA and CGPA calculations.
* Maintain student academic records efficiently.
* Generate detailed academic performance reports.
* Reduce manual calculation errors.
* Improve accuracy, transparency, and efficiency in result management.

## Key Features

* Student registration and profile management.
* Course and credit-hour management.
* Grade entry and update facility.
* Automatic SGPA calculation for each semester.
* Automatic CGPA calculation for overall performance.
* Academic report generation and printing.
* Search and view student records.
* User-friendly administrative dashboard.

## Benefits
