# 🎓 CGPA Calculator (UOH Grading System)

A simple C++ console-based CGPA Calculator designed for students of the University of Haripur (UOH).  
It calculates GPA and CGPA based on semester-wise marks and credit hours.

---

## 🚀 Features
- Calculate GPA for each semester
- Calculate overall CGPA
- UOH grading system implementation
- Simple and user-friendly console interface
- Error-free input handling

---

## 🛠️ Built With
- C++
- Basic Programming Fundamentals
- Console Application

---

## 📌 How to Run
1. Open project in any C++ IDE (JetBrains CLion recommended)
2. Compile the code
3. Run the executable
4. Enter subject marks and credit hours

---

## 🧮 Formula Used
CGPA = (Sum of Grade Points × Credit Hours) / Total Credit Hours
